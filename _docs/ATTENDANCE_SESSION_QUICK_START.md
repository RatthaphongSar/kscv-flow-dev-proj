# ระบบการเข้าเรียน - Quick Start

## 🎯 คุณสมบัติ

✅ **ตั้งค่าการเข้าเรียน**
- ชื่อหัวข้อ (เช่น "สอบกลางภาค")
- ประเภท (เรียน/สอบกลางภาค/ปลายภาค/สอบย่อย/เก็บคะแนน)
- วันที่เริ่มและสิ้นสุด
- หมายเหตุเพิ่มเติม

✅ **ดูข้อมูล**
- แสดงรายการทั้งหมด
- จำนวนนักเรียนเข้าแล้ว
- สีแตกต่างตามประเภท
- วันที่เป็นภาษาไทย

✅ **จัดการ**
- เพิ่ม/แก้ไข/ลบ
- มี confirmation เมื่อลบ
- Validation errors

---

## 👨‍🏫 วิธีใช้ (สำหรับครู)

### 1. เข้าหน้าการเข้าเรียน
```
Class Page → Attendance Tab → ปุ่ม "ตั้งค่า"
```

### 2. Modal "การเข้าเรียน" ปรากฏ

### 3. คลิก "เพิ่มการเข้าเรียน"

### 4. กรอกข้อมูล
- **ชื่อหัวข้อ**: "สอบกลางภาค" (บังคับ)
- **ประเภท**: เลือก (บังคับ)
- **วันเริ่มต้น**: เลือกวัน (บังคับ)
- **วันสิ้นสุด**: เลือกวัน (ไม่บังคับ)
- **หมายเหตุ**: พิมพ์บันทึก (ไม่บังคับ)

### 5. คลิก "บันทึก"

---

## 📋 ประเภทการเข้าเรียน

| ประเภท | สัญลักษณ์ | เหตุผล |
|--------|---------|-------|
| **เรียน** | 📘 สีฟ้า | บรรยาย/อภิปราย |
| **สอบกลางภาค** | 📊 สีส้ม | Midterm exam |
| **สอบปลายภาค** | 🎓 สีแดง | Final exam |
| **สอบย่อย** | ✏️ สีม่วง | Quiz/Short test |
| **เก็บคะแนน** | 🎯 สีเขียว | Attendance check |

---

## 🔧 การแก้ไข

### ขั้นตอน
1. ค้นหารายการที่ต้องการแก้ไข
2. คลิกไอคอน ✏️ (ดินสอ)
3. ฟิลด์ทั้งหมดสามารถแก้ไขได้
4. คลิก "บันทึก"

### สามารถแก้ไขได้
- ชื่อหัวข้อ
- ประเภท
- วันที่เริ่มและสิ้นสุด
- หมายเหตุ
- สถานะ

---

## 🗑️ การลบ

### ขั้นตอน
1. ค้นหารายการที่ต้องการลบ
2. คลิกไอคอน 🗑️ (ถังขยะ)
3. Modal ยืนยันปรากฏ
4. คลิก "ลบ"
5. ลบสำเร็จ

---

## 📊 ข้อมูลที่แสดง

### สำหรับแต่ละรายการ
```
[สีประเภท] ชื่อหัวข้อ
📅 วันเริ่ม - วันสิ้นสุด
หมายเหตุ (ถ้ามี)
👥 นักเรียนเข้าแล้ว: X คน
```

### นับนักเรียนอย่างไร?
- ระบบนับจากจำนวน Attendance records
- ที่เชื่อมกับ session นี้
- อัปเดตอัตโนมัติ

---

## 💾 API Reference

### ดึงรายการ
```
GET /classes/{classId}/attendance-sessions
```

### สร้างใหม่
```
POST /classes/{classId}/attendance-sessions
{
  "subject": "สอบกลางภาค",
  "type": "midterm",
  "startDate": "2567-11-24",
  "endDate": "2567-11-25",
  "description": "..."
}
```

### อัปเดต
```
PATCH /classes/{classId}/attendance-sessions/{sessionId}
{ ... }
```

### ลบ
```
DELETE /classes/{classId}/attendance-sessions/{sessionId}
```

- **วันเริ่มต้น**: วันปัจจุบัน
- **วันสิ้นสุด**: ว่าง (ไม่บังคับ)
- **สถานะ**: active
- **ประเภท**: lesson

## 🎨 ตัวอย่างการตั้งค่า

### ตัวอย่าง 1: สอบกลางภาค
```
ชื่อหัวข้อ: สอบกลางภาคภาษาอังกฤษ
ประเภท: สอบกลางภาค
วันเริ่ม: 24-11-2567
วันสิ้นสุด: 25-11-2567
หมายเหตุ: สอบ 1.5 ชั่วโมง
```

### ตัวอย่าง 2: บรรยายสัปดาห์นี้
```
ชื่อหัวข้อ: Unit 5 - Grammar
ประเภท: เรียน
วันเริ่ม: 24-11-2567
วันสิ้นสุด: (ว่าง)
หมายเหตุ: (ว่าง)
```

### ตัวอย่าง 3: เก็บคะแนน
```
ชื่อหัวข้อ: Attendance Check
ประเภท: เก็บคะแนน
วันเริ่ม: 24-11-2567
วันสิ้นสุด: 30-11-2567
หมายเหตุ: Checking attendance
```

## 🐛 Troubleshooting

### ปัญหา: ไม่เห็นปุ่ม "ตั้งค่า"
**วิธีแก้**: ตรวจสอบว่าเข้าครูบัญชีครูหรือไม่

### ปัญหา: ไม่สามารถบันทึก
**วิธีแก้**: 
- ตรวจสอบกรอกข้อมูล "ชื่อหัวข้อ" และ "ประเภท"
- ตรวจสอบเชื่อมต่อ internet

### ปัญหา: จำนวนนักเรียนไม่ถูกต้อง
**วิธีแก้**:
- รีเฟรชหน้า
- ตรวจสอบ attendance records ในระบบ
