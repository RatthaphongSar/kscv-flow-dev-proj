# 🎉 Chat Features - User Guide

## 📱 What's New

Your chat page now has three powerful new features:

### 1️⃣ @Mention Users
**Type `@` to mention someone in your message**

**Steps:**
1. Click in the message input box
2. Type `@` followed by a name (e.g., `@Mohammad`)
3. A dropdown menu appears showing matching users
4. Click the user you want to mention
5. Their name is inserted with the `@` symbol
6. Finish typing your message and send

**Result:** Mentioned users appear highlighted in yellow with their name in blue

**Example:**
```
Input:  "Hey @Mohammad Ali can you review the doc?"
Output: Hey @Mohammad Ali can you review the doc?
        (Mohammad Ali appears in yellow highlight + blue text)
```

---

### 2️⃣ Attach Files & Images
**Click the paperclip 📎 icon to attach files**

**Supported files:**
- 🖼️ Images: JPG, PNG, GIF
- 📄 Documents: PDF, Word (.doc, .docx), Text (.txt)

**Steps:**
1. Click the **📎 paperclip button** in the message input
2. Choose a file from your computer
3. The file preview appears below the input box
4. Review the filename and size
5. Click **Send** to send the message with attachment
6. (Optional) Click **✕** to remove the attachment

**Result:** 
- 🖼️ **Images** appear as full-width preview in the message
- 📄 **Documents** appear as downloadable file preview with icon

**Example:**
```
Before:    Message input + Click paperclip → File picker
After:    [lecture-notes.pdf - 1.2MB] ✕
          Send button
Result:   Message sent with file attached visible in bubble
```

---

### 3️⃣ File Size Limits
**Maximum 10MB per file**

**Automatic validation:**
- If your file is **too large**, you'll see an error message
- Error shows the actual file size (e.g., "File size: 15.32MB")
- Simply choose a smaller file and try again

**Error Message:**
```
⚠️ File size exceeds 10MB limit. (15.32MB)
```

## ✅ Best Practices

**✅ DO:**
- Use @mentions to get someone's attention
- Send homework/assignment files via chat
- Ask for feedback on documents
- Share notes and study materials
- Keep files under 10MB (compress if needed)

**❌ DON'T:**
- Send files larger than 10MB (will be rejected)
- Spam mentions (only mention when necessary)
- Send multiple large files at once
- Forget to check file size before sending

## 🐛 Troubleshooting

| Problem | Solution |
|---------|----------|
| Mention dropdown not showing | Make sure you typed `@` followed by a name |
| Can't attach a file | Check file type (JPG, PNG, PDF, DOC, TXT only) and size (max 10MB) |
| File attachment disappeared | Click the X button or reload the page |
| Mentioned person not highlighted | They're mentioned if they appear in the dropdown list |
| Message won't send | Make sure you typed something (text or attached a file) |

## 📊 File Size Reference

**Common file sizes:**
- 📸 Photo (JPG): 2-5 MB typical
- 📝 Word document: 0.5-2 MB typical
- 📊 PDF presentation: 1-5 MB typical
- 🎥 Video: Usually TOO LARGE (don't attach videos)
- 🎵 Audio: Usually TOO LARGE (don't attach audio)

**Quick math:**
- 1 MB = 1,024 KB
- 10 MB = 10,240 KB
- Max file size = 10 MB

---

## 💡 Pro Tips

1. **Compress before sending**
   - Large images? Use online compressor
   - Big PDF? Remove unnecessary pages
   - Word doc? Convert to PDF for smaller size

2. **Multiple files?**
   - Send one at a time (one file per message)
   - Or compress them into a .zip file

3. **@mention tips**
   - Use sparingly (only when you need their attention)
   - They'll get notified when mentioned
   - Great for asking specific questions

4. **Share effectively**
   - Add a message explaining what you're sharing
   - Example: "Here's the assignment PDF - due next week"
   - Not just: *sends file*
